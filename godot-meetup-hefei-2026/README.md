# A crash course in multiplayer

Event: [Godot Meetup 合肥 2026]

Playing multiplayer games is fun. Making them - not so much. Part of that can
be attributed to netcode being mystified and difficult to approach.

This talk discusses the challenges, some architectural approaches, and
introduces [netfox].
